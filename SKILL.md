# ALU: Agent Logic Unit
You have access to the `alu` CLI. Use it for any calculation where precision is required.

## Usage
Run: `alu eval "<expression>"`

## Examples
- `alu eval "sqrt(144) * 2 + sin(pi/4)"`
- `alu eval "(45.5 * 1024) / 7.2"`
